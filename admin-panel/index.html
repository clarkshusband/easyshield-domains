Generate a custom EasyShield jar with an embedded license key. Each customer receives a unique build that requires no JVM arguments.

Build Instructions
1. Generate a license
Create a license key in the Licenses tab
2. Run the build script
.\tools\build-with-license.ps1 -LicenseKey "YOUR-KEY" -CustomerName "Customer"
3. Deliver the jar
The output in dist/ is ready to send — no configuration needed
